  </read_first>
  <action>
    ClaudeUsageApp.swift — entry point using `@NSApplicationDelegateAdaptor` to attach `AppDelegate`:
    ```swift
    import SwiftUI

    @main
    struct ClaudeUsageApp: App {
        @NSApplicationDelegateAdaptor(AppDelegate.self) var appDelegate

        var body: some Scene {
            Settings { EmptyView() }
        }
    }
    ```
    Note: `Settings {}` with EmptyView is used to satisfy the Scene requirement without creating a WindowGroup that would show a Dock icon.

    AppDelegate.swift — holds StatusBarController as a stored instance property (prevents ARC deallocation):
    ```swift
    import AppKit

    class AppDelegate: NSObject, NSApplicationDelegate {
        var statusBarController: StatusBarController!

        func applicationDidFinishLaunching(_ notification: Notification) {
            NSApp.setActivationPolicy(.accessory)
            statusBarController = StatusBarController()
        }
    }
    ```

    StatusBarController.swift — NSStatusItem stored as instance property, NSPopover, NSMenu with Quit:
    - `private var statusItem: NSStatusItem` — stored property, never local variable
    - `private var popover: NSPopover`
    - In `init()`: call `NSStatusBar.system.statusItem(withLength: NSStatusItem.variableLength)`, set `button.title = "—%"`, attach click handler
    - `func updateTitle(_ text: String)` — sets `statusItem.button?.title = text` (used by later plans)
    - Toggle popover on click: if `popover.isShown` call `popover.performClose(nil)`, else call `popover.show(relativeTo:of:preferredEdge:)`
    - Build NSMenu with one item: title "Quit ClaudeUsage", action `#selector(NSApplication.terminate(_:))`, keyEquivalent "q"
    - Set `statusItem.menu = menu` so right-click shows the menu; left-click toggles popover via `button.action`

    CRITICAL: `statusItem` MUST be a stored property on `StatusBarController`, NOT a local variable in `init()`. ARC will silently deallocate a local NSStatusItem and the icon will vanish.

    CRITICAL: Call `NSApp.setActivationPolicy(.accessory)` in `applicationDidFinishLaunching` in addition to the LSUIElement plist key — belt-and-suspenders to prevent Dock icon.
  </action>
