  <files>ClaudeUsage/Store/UsageStore.swift</files>

  <action>
Add a private `pollingTask: Task<Void, Never>?` stored property to UsageStore.

Add `startPolling()`:
- Cancels any existing pollingTask first (idempotent — safe to call twice)
- Stores a new unstructured Task that loops: call fetchUsage(), then Task.sleep(for: .seconds(60))
- Loop exits when the Task is cancelled (Task.sleep throws CancellationError on cancel)
- Log each poll iteration at OSLog .debug level: `logger.debug("Polling: fetching usage")`

Add `stopPolling()`:
- Calls `pollingTask?.cancel()` then sets `pollingTask = nil`

The loop structure to use:
```swift
pollingTask = Task { @MainActor in
    while !Task.isCancelled {
        logger.debug("Polling: fetching usage")
        await fetchUsage()
        do {
            try await Task.sleep(for: .seconds(60))
        } catch {
            break  // Task cancelled during sleep
        }
    }
}
```

Fix error display in `fetchUsage()` failure branch:
- `.apiUnauthorized` case → keep "Auth expired" (no change)
- `default` case → change from "API unavailable" to "—" (em dash, no percent sign)
  Locked decision: "—" distinguishes API error from "—%" loading state (DISP-01)

No exponential backoff. No retry acceleration. Plain 60-second cadence regardless of errors.
